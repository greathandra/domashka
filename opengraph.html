Open Graph

Facebook / Meta Протокол мета-тегов для управления превью ссылок в соцсетях.

Что это?

Open Graph (OG) — протокол, созданный Facebook. Мета-теги с префиксом og: размещаются в <head> и описывают, как ссылка должна выглядеть при публикации. Используется Facebook, VK, Telegram, LinkedIn и многими мессенджерами.

Обязательные теги

ТегНазначение
og:titleЗаголовок превью
og:descriptionКраткое описание
og:imageURL изображения (рекомендуется 1200×630)
og:urlКанонический URL страницы
og:typeТип: website, article, product...

Пример разметки

<meta property="og:type" content="article">
<meta property="og:title" content="Заголовок страницы">
<meta property="og:description" content="Описание для превью">
<meta property="og:url" content="https://example.com/article">
<meta property="og:image" content="https://example.com/preview.jpg">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:locale" content="ru_RU">
<meta property="og:site_name" content="Мой сайт">
Совет: После изменения OG-тегов сбросьте кэш через Facebook Sharing Debugger.
