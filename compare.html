Сравнение форматов

Когда использовать Schema.org, JSON-LD, Open Graph и Twitter Cards.

Критерий Schema.org JSON-LD Open Graph Twitter Cards
Назначение Словарь типов Формат записи Превью в соцсетях Карточки в X
Где размещается Любой формат <script> <meta property> <meta name>
Для поисковиков Да Да Частично Нет
Для соцсетей Нет Нет Да Да
Рекомендация Google Основа Предпочтительно — —

Что использовать на практике?

Для SEO

JSON-LD + типы Schema.org под ваш контент (Article, Product, FAQ...).

Для соцсетей

Open Graph на всех страницах + Twitter Cards для X.

Итог: Используйте все форматы вместе. JSON-LD для поиска, OG + Twitter для шаринга. Это не взаимоисключающие технологии.
